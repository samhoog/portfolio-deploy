GradeBoost

Timeline

2023 - 2024

Team

Tech Stack

Angular, TypeScript, Docker, Supabase, Resend

Overview

GradeBoost offers a centralized, user-friendly interface that allows students to submit, track, and manage grade appeals effortlessly. When a project is graded incorrectly or you resubmit a lab, the process can be down quickly and effectively though GradeBoost, instead of cluttering up email inboxes.

Features

With features like professor, student and grader roles with corresponding dashboards for overseeing all appeals, as well as live chats for seamless communication, both students and educators can engage effectively.

Using Resend we implemented weekly digest emails and adjustable notification settings, as well as having on-premise single sign-on authentication. Plus, pre-written response templates help professors save time while focusing on supporting student success.
